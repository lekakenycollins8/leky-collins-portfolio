## Overview

TutorsHub is a cutting-edge platform designed for educators to build personalized portfolios, showcase expertise, and connect with students. Featuring a sleek, responsive UI, TutorsHub makes it easy for tutors to manage their availability and interact with learners.

## Key Features

- **Modern UI/UX**: Built with Material-UI and Tailwind CSS for a seamless user experience.
- **Dynamic Blog System**: Tutors can share educational content and insights.
- **Contact Management**: Integrated contact form for student inquiries.
- **Floating Navigation**: Provides a smooth, accessible navigation experience.
- **Portfolio Customization**: Enables tutors to highlight their expertise and experience.
- **Responsive Design**: Fully optimized for all device sizes.

### Dynamic Navigation

- Floating navigation bar adapts to screen size.
- Smooth scrolling and animations.
- Consistent navigation across all pages.

### About Page

- Interactive table of contents with smooth scrolling.
- Fixed sidebar navigation with automatic highlighting of active sections.
- Professional experience timeline with detailed personal and work history.

### Contact System

- Professional contact form with validation.
- Formspree integration for message delivery.
- Real-time form state management with success/error handling.
- Animated submit button with loading states.

### UI Components

- Animated background effects and dynamic line patterns.
- Custom buttons with animated gradients.
- Responsive typography with gradient text effects.
- Material-UI integration for cohesive styling.

### Page Layouts

Each page is designed with:
- Custom background effects.
- Responsive layouts and SEO optimization.
- Consistent navigation and content-specific styling.

## Tech Stack

- **Frontend**: Next.js 14 with App Router
- **Styling**: Material-UI + Tailwind CSS
- **Animations**: Framer Motion
- **Forms**: Formspree Integration
- **Icons**: React Icons
- **Typography**: Optimized with `next/font`

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/lekakenycollins8/TutorsHub.git
   cd TutorsHub
   ```
2. Install dependancies
  ```sh
   npm install
   # or
   yarn install
   ```
3. Run the development server:
```sh
  npm run dev
  # or
  yarn dev
```
4. Open http://localhost:3000 to view the application.
