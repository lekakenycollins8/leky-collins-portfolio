## Overview

Fitness Dashboard is an intuitive and user-friendly platform that enables users to browse exercises based on muscle groups, difficulty levels, and more. It empowers fitness enthusiasts with a seamless way to discover new workouts, save their favorites, and access exercise details anywhere.

## Live Demo

[Visit Fitness Dashboard](https://fitness-dashboard.vercel.app/)

## Features

- **Exercise Categories**: Browse exercises by muscle groups (abs, biceps, triceps, etc.) and difficulty levels.
- **Smart Search**: Quickly find specific exercises with real-time search functionality.
- **Favorites System**: Save preferred exercises for quick access later.
- **Responsive Design**: Works seamlessly across phones, tablets, and desktops.
- **Detailed Instructions**: Each exercise comes with clear, step-by-step guidelines.

## Why Use Fitness Dashboard?

Whether you're a beginner or an experienced athlete, Fitness Dashboard helps you:

- Discover new exercises for various muscle groups.
- Learn proper techniques with detailed exercise instructions.
- Create a personalized collection of favorite exercises.
- Access workouts on any device with a responsive design.

## Getting Started

Clone the repository and run the project locally:

```bash
git clone https://github.com/lekakenycollins8/Fitness-Dashboard.git
cd Fitness-Dashboard
npm install
npm run dev
```

## How It Works

- Browse exercises by category or use the search feature.
- Click the heart icon to save exercises to your favorites.
- Access saved exercises anytime in the Favorites section.

## Tech Stack

- **Frontend**: React
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Data Fetching**: React Query
- **Type Safety**: TypeScript
- **Development**: Vite for fast performance

## API Integration

This project integrates with the Exercise API to fetch comprehensive workout data, including:

- Exercise names and descriptions
- Targeted muscle groups
- Difficulty levels
- Step-by-step instructions

API authentication is managed using environment variables:

```env
VITE_EXERCISES_API_KEY=your-api-key
VITE_EXERCISES_API_URL=https://api.api-ninjas.com/v1
```

To run the project locally, obtain an API key and configure your `.env` file accordingly.

## Available Pages

- **Home**: Featured strength exercises
- **Categories**: Browse exercises by muscle group and difficulty
- **Search**: Find specific exercises
- **Favorites**: View and manage saved exercises
