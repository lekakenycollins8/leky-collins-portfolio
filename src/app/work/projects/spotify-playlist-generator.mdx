## Overview

The **Spotify Playlist Generator** is a web application designed to enhance users' music discovery experience by allowing them to view their top tracks across different time ranges and create custom playlists. Through seamless Spotify OAuth authentication and an intuitive interface, users can explore their listening history and easily curate new playlists.

## Getting Started

### Prerequisites
- **Node.js** (v14 or higher)
- **npm or yarn**
- **Spotify Developer Account**

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/lekakenycollins8/spotify-playlist-generator.git
   cd spotify-playlist-generator
   
   # Install client dependencies
   cd client
   npm install
   
   # Install server dependencies
   cd ../server
   npm install
   ```

2. Create a `.env` file in the `server` directory with your Spotify credentials:
   ```env
   CLIENT_ID=your_spotify_client_id
   CLIENT_SECRET=your_spotify_client_secret
   REDIRECT_URI=http://localhost:8080/auth/callback
   FRONTEND_URI=http://localhost:3000
   ```

3. Start the development servers:
   ```sh
   # Start the backend server
   cd server
   npm run dev
   
   # Start the frontend development server
   cd client
   npm run dev
   ```
